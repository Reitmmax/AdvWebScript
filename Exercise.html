Max's Exercises

This page will be showcasing exercise I have done throughout the semester!

Mod 1

Pig

This exercise is called "Pig". The goal of the game is to get to 10 points. It is a 2 player game. You click to roll and add the total you get for each roll, if you roll a 1, you lose the points you get then pass the turn to the next player. You can choose to "hold" which allows you to pass the turn and keep all your points.

This teaches how to use numbers in JavaScript code. It also reteaches loops, how to set constants and variables, and how to change the variables within the loop.

Countdown

This exercise is called "Countdown". This is a timer that tells how long (or long ago) a date is.

This teaches how to put restrictions on information and how to keep it validated, and change responses depending on what was typed in the textboxes.

Mod 2

Swapper

This is a page with images on it. Whenever you click the image on top, it changes the image and the title of the image you are looking at. It does this by using a tags and switching the src of the page to whatever is in the href, and changes the caption to whatever the title is set to on the image.

Tabs

Whenever the header is clicked, it toggles .show and .hide. When clicked at first, it changes the + to - and displays text below the tab. Whenever clicked again, it changes - back to + and hides the text it displayed.

Mod 3

Slideshow

When you first enter the page, the images start going through a slideshow. During the slideshow, the images fade in and out. If you want to look at an image for longer, you can click on the image to stop the slideshow, then you can click it again to start the slideshow up again.

Flowers

Whenever you scroll on this page, the title disappears and reappears when you scroll back to the top. You'll also notice that the body text is also animated, but whenever its scrolled away from, it does not reset the animation. If you were to click my name on the footer, you will see a little emoji slide out and the arrow pointing changes. If you click it again, it hides the emoji.
