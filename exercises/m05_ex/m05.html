m05_1

			
// JavaScript Document
"use strict";

const $mess1 = 'These are words. Some more words.';
const $mess2 = 'This is a phrase. A very short sentence. A slightly kind of maybe long sentence.';

const $parag = ($part) => {
  console.log(`This is a paragraph made of different stuff. ${$part}`);
};

$parag($mess2);

This code has 3 constants, $mess1, $mess2 and $parag. mess1 and mess2 are simply just strings. parage is using $part to add one of the two constants above to its code. In this case, I have it calling $mess2. The message should display: "This is a paragraph made of different stuff. This is a phrase. A very short sentence. A slightly kind of maybe long sentence." because it is combining them in $parag($mess2).

M05_2

		
		// JavaScript Document
"use strict";

const $mess1 = 'These are words. Some more words.';
const $mess2 = 'This is a phrase. A very short sentence. A slightly kind of maybe long sentence.';

const $messages = [$mess1, $mess2];

var $states = 1;

const $rating = ($states > 2) ? 'There are a lot of statements here.' : 'This is a fairly short statement.';

const $parag = ($part) => {
  console.log(`This is a paragraph made of different stuff. ${$part}`);
};

let i = 0;
while (i <= 1) {
$parag($messages[i]);
$states++;
i++;
}

console.log($rating);

This is the second part of the project. This time, we decided to put $mess1 and $mess2 into an array. We also made a new varible called "$state" and a new constant called "$rating". Essentially, what this code is doing is calling the array, then displaying what is in that part of the array, then calls the next part, and so on. When its done, it calls $rating and displays "This is a fairly short statment" because states should be equal to 3. If it was less than 2, it would instead display: "There are a lot of statements here"
